# Awesome Paper Note

> 计算机体系结构领域论文的深度阅读笔记，重点聚焦DRAM方向。

## 简介

本仓库收录计算机体系结构（Computer Architecture）领域的高质量论文精读笔记。每篇笔记均包含论文的核心思想提炼、技术细节剖析、实验方法解读以及审稿人视角的批判性评价。

所有笔记使用**中文**撰写，专业术语保留英文原文。

## 目录结构

```
.
├── FUN/                 # 好文章 or 新文章
├── PAGE_POLICY/         # DRAM 页面管理策略
├── MC_DSE/              # 内存控制器设计空间探索
├── MEMORY_SCHEDUING/    # 内存调度策略
└── note-prompt.md       # 笔记撰写模板与规范
```


## 笔记格式

每篇笔记遵循统一模板（见 [note-prompt.md](note-prompt.md)），包含以下章节：

1. **基本信息** — 标题、作者、发表会议/期刊、年份
2. **一句话总结** — 论文核心贡献的高度概括
3. **问题与动机** — 论文要解决的问题及研究背景
4. **核心方法** — 关键洞察、技术细节、与已有工作的区别
5. **实验评估** — 实验设置、主要结果、结果分析
6. **审稿人视角** — 优点、不足、开放性问题
